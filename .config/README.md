### How to setup

Make sure to create backup.

- Clone the repo
- Copy files into your `~/.config` folder.
- You can edit the keybinds of `hypr/hyprland.conf` according to your keybinds.

### Some useful alias

- `update` -> updates your system
- `uwu` -> basic alias for `sudo`
- `uwu install <package_name>` -> installs a package via `pacman`
- `..` `...` `....` -> these are alias for: `cd ..` `cd ../..` `cd ../../../` respectively.
- `rice` -> `nano ~/.config/hypr/hyprland.conf` You can change it to vim or nvim or emac whatever you like.
- `pac <package_name>` -> `sudo pacman -S`
- `uwur <package_name>` -> `sudo pacman -Rns` To remove a pkg
- `y <package_name>` -> `yay -S`
- `c` -> `clear`
- `q` -> `exit` To exit terminal
- `ls` -> Normal List
- `la` -> List all including hidden
- `ll` -> same as la
